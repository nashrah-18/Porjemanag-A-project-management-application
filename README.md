# Projemanag

*A Project Management Application*

**Projemanag** is a modern project management application designed to streamline **planning, task organization, and team collaboration**.
This project earned ** 2nd Prize** during my **final year of BCA**, recognizing its functionality, design, and real-world applicability.

---

## Overview

Projemanag helps individuals and teams manage projects efficiently using **custom boards, task lists, and collaborative tools**—all within a clean and intuitive Android application.

---

## Key Features

### Project & Task Management

1. **Customizable Boards**
   Create personalized dashboards to organize projects and workflows.

2. **Task Assignment**
   Create, assign, and manage tasks with ease.

3. **Task Prioritization**
   Categorize and prioritize tasks using flexible lists.

4. **Drag-and-Drop Support**
   Reassign tasks effortlessly for better workflow management.

---

### Team Collaboration

5. **Member Management**
   Add and manage team members with defined roles and permissions.

6. **Collaboration Tools**
   Seamless team interaction using **cards and lists**.

---

### Personalization & Sync

7. **Custom Board Appearance**
   Personalize boards with custom backgrounds and logos.

8. **Seamless Synchronization**
   Real-time data sync across devices—access your projects anytime, anywhere.

---

## Tech Stack

* **Kotlin (XML)**
  Used for Android application development, ensuring high performance and stability.

* **Firebase**
  Provides secure authentication, real-time database synchronization, and cloud storage.

* **Android SDK**
  Core development toolkit for building the Android application.

---

## Use Cases

* Personal task management
* Corporate project management (e.g., Marketing teams)
* Educational & research project management
* Event planning
* Software development team management
